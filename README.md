# Vue 3 + TypeScript + Vite

This template should help get you started developing with Vue 3 and TypeScript in Vite.

Project setup -
1. Clone the project by running the command - git clone https://github.com/Ranjeet-vishwakarma-8081/Smart-Todo-App.git
2. Install all the development dependencies - npm install
3. Run your project - npm run dev
